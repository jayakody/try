# Disconnect-CohesityCluster

## SYNOPSIS
Disconnects from the Cohesity Cluster.

## SYNTAX

```
Disconnect-CohesityCluster [<CommonParameters>]
```

## DESCRIPTION
Disconnects from the Cohesity Cluster and invalidates the session.

## EXAMPLES

### EXAMPLE 1
```
Disconnect-CohesityCluster
```

Disconnects from the Cohesity Cluster and invalidates the session.
